Functions

Functions can feel tricky and difficult for beginners, but if you stick with it, you will catch on faster than you think. A JavaScript function is defined with the 'function' keyword, followed by a name(which is applied to the function), followed by parentheses (). Inside the parentheses are what we call 'parameters', which are the names listed in the function definition. You can have more than one parameter if necessary, just remember to seperate multiple parameters with commas. finally, the code which is to be executed is inside a pair of curly brackets {}. below is an example of a simple function: 

                        
      function myFunc(){ 
      console.log('hello world');
      }

As you can see, our function is named 'myFunc', doesnt take any parameters, and will return "hello world" when the function is called. To call our function, we would have to call it by its name, which in this case is myFunc().

Objects

Objects in Javascript are similar to objects in other programming languages. These 'objects' seem complex and while they can be, are actually very similar to objects in our every day life. If you can think of javascript objects as a regular object in real life, it will be a lot easier for you to grasp the concept. In javascript, an object is an entity with 'properties' and 'type'. In real life, a cup has a color, a design, weight, and material its made from. Javascript objects can be thought of similarly. Below you will find a simple javascript object: 

                        
let myHonda = {
        color: 'red',
        wheels: 4,
        engine: {cylinders: 4, size: 2.2}
       }
In this object, we have 3 Key-Value pairs. The first is 'Color:red', where the 'Key' is 'Color' and the value is the string 'red'. the next Key-Value pair is 'Wheels:4', where the 'key' is wheels and the value is the number 4. Finally, we have another object in our third key-value pair, which consists of 2 elements. Nested objects are very simple to understand once you get enough practice.

Arrays

In Javascript, we can use variables to store values. But what if we wanted to store multiple values in one single variable? Welcome to Arrays. These special variables, can hold more than one value at a time. These variables are helpful when you have lists you want to store. They seem redundant at first, but we have to realize, some lists can be hundreds, even thousands of values long. With an array, searching for a certain value would take less than a second. Arrays have a name, and store their values in a set of brackets [], the elements inside are seperated by commas. 

                    

     let myCars= ['charger', 8, 'camry', 4, 'accord', 6]
In the array above, named 'myCars', we have 6 elements inside. 3 string type, and 3 number type. Arrays can store different types of values, including booleans, strings, numbers, and even objects. Each element in an array has an 'index', which is the number assigned to its position within the array. in our myCars array, 'camry' is being held in index:2. This means console.log(myCars[2]) would print out 'camry'. Arrays can be a simple concept to grasp at first, but can get a lot more complex. They are a very powerful tool.
